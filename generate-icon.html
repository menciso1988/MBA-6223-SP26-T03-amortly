Amortly Icon Generator

Generate the PNG icons needed for the iPhone home screen app icon.

512 × 512
180 × 180 (iPhone)
60 × 60 (preview)

After downloading both files:

  1. Move both PNG files into the amortly/icons/ folder
  2. Open your terminal in the amortly folder and run:
    git add icons/ && git commit -m "Add app icons" && git push
  3. On your iPhone, open Safari and go to your GitHub Pages URL
  4. Tap the Share button (box with arrow) → Add to Home Screen
  5. Tap Add — Amortly will appear on your home screen with this icon
